Computer resources

Matlab/Octave Scripts

Matlab can be accessed in the open access labs on campus (e.g. in Blocker) or through the Texas A&M Virtual Online Access Lab. The scripts below can also be used in GNU Octave, which is available for free download here.
  • Script for plotting a direction field: DirectionField_MATH308.m
  • Script for plotting the solution of a spring-mass problem (Sections 3.7 and 3.8): MechanicalVibrations_MATH308.m (examples done in class: here)
  • Script for plotting a phase portrait: PhasePlane_MATH308.m
  • Script and functions for computing an approximation of a 1st order DE: main.m , forward_euler.m , backward_euler.m , crank_nicolson.m , heun.m

Java Programs - Courtesy of John C. Polking, Rice University

Helpful information about downloading and running these programs can be found here.
  • Program for plotting a direction field and solution curves: dfield.jar
  • Program for plotting a phase portrait: pplane.jar
